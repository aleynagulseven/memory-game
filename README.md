# memory-game
memory-game JS
Memory game is a card matching game that you flip two cards and try to find the same pictures. Game ends when you match all the cards.
This game includes these JavaScript functions :
for loops
addEventListener
setAttribute
getAttribute
data-name
document.querySelector
createElement
